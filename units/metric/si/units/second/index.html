The Second


Fact File
Unit Name Second
Unit Symbol s
Quantity Name Time
Quantity Symbol t
Defining Constant(s) The Unperturbed Ground State Hyperfine Transition Frequency of The Caesium-133 Atom (ΔνCs)
Unit System International System of Units (Système International d'Unités), SI

SI Definition

"The second, symbol s, is the SI unit of time. It is defined by taking the fixed numerical value of the caesium frequency, ∆νCs, the unperturbed ground-state hyperfine transition frequency of the caesium 133 atom, to be 9 192 631 770 when expressed in the unit Hz, which is equal to s-1."
